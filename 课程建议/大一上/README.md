## 大一上

### 高学分优先
- 高等数学A（一）—— 3学分
- 大学英语（一）—— 3学分
- 思想道德与法治—— 3学分
- 线性代数B—— 3学分

**学习策略：**
- 高数：刷往年题 + 吃透课本例题。不要只听课，题量要够。
- 线代：同上，题型相对固定，往年题是捷径。
- 大英：准备四级的时候顺带就复习了

### 水课怎么处理

思修考试书上都有，重点是报告，一定要好好写，认真去实践，不然会非常拉分

### 微电子学导论

1学分的概论课，本质上是专业介绍性质。

如果用了AI辅助，记得改一改句式，加点自己找的图，去掉AI味。直接复制粘贴很容易被识别。

### 课余
大一上无法报名四级，但是单词一定要坚持每天背一点。C语言非常重要，一定要提前自学，至少要到指针。Python也要提前学一点，不仅仅是为了应付下学期Python，更重要的是培养逻辑思维能力。
