## <span id="why-microelectronics">你为什么选择微电子</span>

这个问题是所有事情的起点。

先说一下这个专业的现实状况：

**专业隶属于物理科学与技术学院**，不是电子与信息工程学院。学院的底子在物理和材料方向。培养方案的课程设置也反映出这一点：半导体物理、固体物理、量子力学、薄膜材料——底子是物理，不是电路。

**专业太新了。** 微电子是2024年新专业，目前还没有毕业生（截止至2026.6）。没有上一届的就业数据，没有学长学姐的考研经验，没有已经走过一遍的人告诉你坑在哪。你做的所有判断，都是在没有样本的情况下自己摸索出来的。

**扩招。** 第一届1个班，第二届3个班，第三届4个班。

**转专业的人涌进来。** 微电子现在是一部分人的转专业目标，转过来却和我们一样面临模糊的未来。

这些事情指向一个问题：**你确定你是真的想做微电子，还是只是被"微电子"这三个字带进来的？**

如果你是被"芯片"这个词吸引进来的，那你要知道：芯片设计和微电子本科之间，隔着一整层的知识缺口。学校教的是半导体物理和薄膜材料，不是数集和模集，不是C语言和Verilog。你不自己补，毕业的时候离"芯片"还是很远。

如果只是想拿个工科学位，那跟着培养方案走也能毕业。半导体行业不只有设计，工艺、制造、封装、测试也需要人。

如果你发现自己根本不想学硬件、无法坚持，转专业去电子信息也可以考虑。

**想清楚这个问题，后面的所有判断才有意义。**


## <span id="three-layers">微电子的三层结构</span>

微电子有三层：

1. **设计/系统**（上层）—— 数集、模集、Verilog、版图、架构
2. **工艺/制造**（中层）—— 薄膜、光刻、掺杂、封装
3. **材料/物理**（下层）—— 半导体物理、器件物理、表征

**大多数普通学校的微电子本科容易滑到下两层。**——因为好教、有实验设备、能毕业。


## <span id="syllabus-analysis">苏科大教到了哪一层</span>

基于2023版培养方案：

**有的：** 半导体物理、固体物理、量子力学、薄膜材料、半导体制造技术、集成电路原理与设计、EDA技术

**没有的：** 数集、模集、Verilog、C语言、FPGA、版图设计、流片实践


## <span id="undergrad-is-not-enough">"本科本来就教不完"</span>

这句话来自IC指南。写的人在复旦。

复旦有数集、模集、Verilog、流片，课比我们多得多——但作者依然说"在围城里爱过、恨过、挣扎过"。

> "本科打基础，硕士算入门，博士顶多叫略懂。"

不是复旦的课不够，是这门学科本身就超出了本科的范围。

**所以缺课这件事，不完全是学校的责任。但你不能用学校的限制，来定义自己的上限。**


## <span id="push-exam-mismatch">保研与设计的错位</span>

"如果你觉得读书就是为了模仿别人的想法，那么这是思想上的懒惰。请丢开书本自己思考。"

推免的规则上写：**综合成绩 = 学业成绩 × 80% + 综合素质 × 20%**。学业成绩看加权平均分，综合素质靠竞赛、论文、专利等加分。

这套规则本身没有问题。但存在着结构性的错位：

电子方向（设计/系统）本科阶段出成果比较难。这意味着如果你走电子方向，综测那20%大概率拿不到多少加分。

而其他方向——比如材料/器件/工艺——相对容易出成果。不是说你随便做做就能发论文，而是从时间周期和产出概率上看，确实比电子方向更容易在本科阶段拿出东西来。

**保研可以走，但你要清楚这条路大概率通向哪里。你拿到保研资格之后，去的是不是你真正想去的地方？**

**这不是在否定保研。但我想指出"保研"和"做IC设计"是两件不同的事。走电子方向，就要接受本科阶段可能没什么看得见的成果；想要保研加分，就要接受可能被推向保研论文产出的方向。大部分人难以兼得，要做取舍。**


## <span id="about-employment">关于就业</span>

微电子专业是**2024年新专业**，目前还没有毕业生（截止至2026.6）。没有就业数据。

官方给出的就业去向是"微电子、光电材料与器件、集成电路制造工艺与测试等产业"。换句话说，学校能对接的岗位主要在工艺和制造端。

但苏科大有一个其他同档次学校没有的优势：**在苏州**。苏州是国内集成电路产业的重镇，本地有晶方半导体等企业。你在苏州上学，意味着你有机会接触到比其他城市更多的实习和产业资源。

所以现实是：**学校的牌子不会帮你太多，但城市的位置可以帮你一些。** 能不能用上这个优势，看你自己的行动。


## <span id="paths">几条路</span>

以下几条路，各自需要的东西不一样，**不可能全抓**。

**1. 考研**
核心是初试分数。适合愿意系统准备考试的人。不用卷绩点。

**2. 保研**
核心是绩点排名 + 项目经历/竞赛加分。扩招后名额稀释，绩点不够一切归零。

**3. 竞赛 + 项目**
参与专业相关的竞赛，积累项目经验。核心价值不在于拿奖，在于让你在过程中真正动手，积累调电路、写代码、debug的经验。这些东西保研复试能聊，考研复试能聊，找工作也能聊。**项目经验是唯一贯穿所有出路的共同语言。**

**4. 跟着走**
不挣扎、不补课、完全跟着培养方案走。
结果：毕业时懂半导体物理、懂薄膜材料、懂制造工艺，不会数集、模集、Verilog、FPGA。去向是半导体工艺/制造/封装方向。


## <span id="advice">建议</span>

**有明确追求的人，不必被任何建议绑住。** 如果你已经知道自己要做什么、并且正在做——那就继续。

**物理 + 电子 + 竞赛 + 考研** 这条路很难，也没有人试过，请仔细考虑。

剩下的路，只能你自己走。
